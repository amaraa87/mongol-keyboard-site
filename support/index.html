Гар цэсэнд харагдахгүй байна The keyboard doesn't appear

Settings → General → Keyboard → Keyboards → Add New Keyboard… → Монгол Keyboard-г нэмсэн эсэхээ шалгаарай. Нэмсний дараа аль ч апп дотор 🌐 товчийг дараад (эсвэл удаан дарж) Монгол Keyboard-г сонгоно.

Add the keyboard in iOS Settings first, then switch to it with the 🌐 key in any app.

“Allow Full Access” яагаад хэрэгтэй вэ? Why Full Access?

Зөвхөн ✨Засах товчны AI дуудлага сүлжээ ашигладаг учраас iOS үүнийг шаарддаг. Бичих, хөрвүүлэх нь бүрэн офлайн — Full Access-гүйгээр ч ажиллана. Бичсэн зүйл тань хэзээ ч бүртгэгддэггүй, илгээгддэггүй — privacy policy-г үзнэ үү.

iOS requires it for any keyboard that makes network requests; we use it only for the ✨Засах call. Typing never leaves your device.

✨Засах ажиллахгүй байна ✨Засах isn't working

Дарааллаар нь шалгаарай:

1 — Гарын тохиргоонд Allow Full Access асаалттай юу?
2 — Апп дотор сервер URL эсвэл API түлхүүр оруулсан уу? (Аль нэг нь л хангалттай.)
3 — Өгүүлбэр кириллээр байна уу? Латин голдуу бичвэрийг Засах илгээдэггүй.
4 — 2 секунд тутамд нэг, өдөрт 200 хүсэлтийн хязгаартай — түр хүлээгээд дахин оролдоорой.

Check Full Access, then that a server URL or API key is set, the sentence is Cyrillic, and you're within the 2s/200-per-day limits.

Засах товч “нэвтрэх ⚙︎” гэж анивчаад байна The fix key flashes “нэвтрэх ⚙︎”

Тохиргоо дутуу гэсэн үг: Монгол Keyboard аппыг нээгээд сервер URL (эсвэл API түлхүүр) оруулаад Хадгалах дарна. Серверийн токен тохируулсан бол зөв эсэхийг шалгаарай.

Open the host app and save a server URL (or API key); if your server uses a token, check it matches.

Ямар үсэг ямар товчоор гарах вэ? What maps to what?

sh→ш · ch→ч · ts/c→ц · kh/h/x→х · yo→ё · yu→ю · ya→я · ii→ий · oe/ö→ө · ue/ü→ү — ө, ү авахын тулд o, u товчийг удаан дарна. Түгээмэл үгсийн эгшгийн алдаа үг дуусахад автоматаар засарна (unoodor → өнөөдөр). Эгшгийн дараах i нь й болно (sain→сайн). АБ↔AB товчоор хөрвүүлэлтийг түр унтрааж латинаар бичиж болно.

Long-press o or u for ө and ү. Common vowel mistypings auto-correct when you finish a word (unoodor → өнөөдөр). The АБ/AB corner key toggles raw Latin typing.

Хэрэглээний хязгаар ба өртөг Limits and cost

Засах нэг удаад нэг л өгүүлбэр (≤500 тэмдэгт) илгээдэг, 2 секунд тутамд нэгээс олонгүй, өдөрт 200-аас илүүгүй. Хариу нь зөвхөн засварласан өгүүлбэр байх боломжтой тул энэ холболтоор чат хийх, эссэ бичүүлэх зэрэг кредит шатаах хэрэглээ бүтэхгүй.

One sentence per tap, min 2s apart, max 200/day — and the response can only ever be the corrected sentence, so the integration can't be repurposed into a chat.
