Зай дарахад үг бүтнээрээ хөрвөнө: sain→сайн (эгшгийн дараах i→й), sh→ш, ts→ц, ухаалаг эгшиг unoodor→өнөөдөр — бүгд төхөөрөмж дээр, сүлжээгүйгээр.

Суулгах / Install

Гурван алхамд бэлэн

  1. Гараа нэм Settings → General → Keyboard → Keyboards → Add New Keyboard… → Монгол Keyboard Add the keyboard in iOS Settings.
  2. “Allow Full Access”-г асаа Зөвхөн ✨Засах товчны AI дуудлагад хэрэгтэй — бичих, хөрвүүлэх нь үүнгүйгээр бүрэн ажиллана. Needed only for the ✨Засах AI call; typing works fully without it.
  3. 🌐 товчоор сонгоод бич Аль ч апп дотор 🌐 товчоор Монгол Keyboard-г сонгоод sain baina uu гэж бичээд үзээрэй. Switch with the 🌐 key and start typing.

✨Засах

AI засвар — таны кредитийг шатаахгүйгээр

Засах товч курсорын өмнөх өгүүлбэрийг л (дээд тал нь 500 тэмдэгт) илгээж, зөв бичгийн алдааг нь засаад буцаана. Энэ холболт зориулалтаараа зөвхөн засвар хийдэг — чат биш:

Бичих үед юу ч илгээгддэггүйЗөвхөн та ✨Засах дээр дарсан үед л нэг өгүүлбэр илгээгдэнэ.
Хязгаар бүхий, хаалттай загвар2 секунд тутамд нэг, өдөрт 200 хүсэлт. Хариу нь зөвхөн засварласан өгүүлбэр байж чадна — эссэ, орчуулга, чат байх боломжгүй.
Түлхүүргүй горимӨөрийн сервер тохируулбал төхөөрөмж дээр ямар ч API түлхүүр хадгалагдахгүй.
