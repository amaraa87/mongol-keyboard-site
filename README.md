# mongol-keyboard-site

Public web pages for the **Монгол Keyboard** iOS app: landing page, privacy
policy, and support — the URLs App Store Connect requires. Published to
GitHub Pages by `.github/workflows/pages.yml` on every push to `main`.

- Landing: `/`
- Privacy policy: `/privacy/`
- Support: `/support/`

The app itself lives in a separate private repository; this repo contains
only the static site. The privacy policy here mirrors the app repo's
`PRIVACY.md` — when one changes, update the other.
