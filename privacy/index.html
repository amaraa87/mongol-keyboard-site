Privacy Policy

The short version: your typing stays on your device, we log nothing, and nothing is ever sent anywhere except the single sentence you explicitly ask to have corrected.

Монгол Keyboard is an iOS keyboard that converts Mongolian Latin transliteration to Cyrillic on your device, with an optional AI grammar-correction feature. This policy describes exactly what the app does with your data.

What the keyboard does with your typing

  • All transliteration (Latin → Cyrillic) happens entirely on your device. Keystrokes are never logged, stored, or transmitted.
  • The keyboard contains no analytics, no advertising SDKs, and no trackers of any kind.

The ✨Засах (grammar fix) feature

  • Only when you explicitly tap the Засах key, the current sentence (at most 500 characters) is sent over an encrypted HTTPS connection for grammar correction, and the corrected sentence is returned.
  • Depending on how you set the app up, that request goes to one of two places:
    • Direct mode — straight to the Anthropic API (api.anthropic.com), authenticated with your own API key.
    • Server mode — to the grammar server whose URL you configured in the app, which forwards the sentence to the Anthropic API. The server holds the API key, enforces rate limits, and never stores or logs sentence text. In this mode the request also carries a random device identifier (a locally generated UUID) used only for rate limiting — it is not linked to your name, account, or anything else about you.
  • Nothing is sent while you type. The request contains only that one sentence — no conversation history, no location, and no identifiers beyond the anonymous UUID described above.
  • The app does not store the sentences it corrects. Processing by Anthropic is governed by Anthropic's privacy policy; with standard API settings Anthropic does not use API inputs to train its models.
  • If you never tap Засах (or never enter an API key or server address), the keyboard makes no network connections at all.

Your API key

  • An API key you enter in the app is stored only on your device, in the app's private App Group container, so the keyboard extension can read it.
  • It is never transmitted anywhere except directly to api.anthropic.com to authenticate your own requests, and it never appears in logs.
  • In server mode no API key is stored on the device at all.

iOS "Allow Full Access"

iOS requires the Allow Full Access permission for any keyboard that makes network requests. Монгол Keyboard uses this permission solely for the Засах call described above. Typing and transliteration work fully without it.

Data we collect

We collect no personal data. In direct mode we (the developer) operate no servers, receive no data, and have no way to see anything you type. In server mode the grammar server processes each request in memory — the sentence is forwarded for correction and then discarded; only anonymous request counts per device UUID are kept for the current day to enforce rate limits. Sentence text is never written to logs or storage.

Children's privacy

The app collects no personal data from anyone, including children.
